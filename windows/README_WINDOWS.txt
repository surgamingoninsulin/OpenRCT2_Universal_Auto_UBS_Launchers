======================================================================
WINDOWS LAUNCHER INSTRUCTIONS
======================================================================

1. Move 'usb_scanner.vbs' into your user path:
   C:\Users\GamingOnInsulin\usb_scanner.vbs
2. This script matches the settings inside your Windows Task Scheduler.
3. When an event trigger with EventID 1006 fires from the 
   'Microsoft-Windows-Partition/Diagnostic' path, Windows will 
   silently run this file to launch OpenRCT2 from your USB drive.
